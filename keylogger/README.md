# Keylogger 

This is a modified version of @[R3x](https://github.com/R3x)'s Keylogger, making use of timers and files to periodically write the logged keys to a file.

**DISCLIMER**: Don't do it.

```

make 
insmod key.ko
cat /proc/keylogger
cat /tmp/log

```
